# Project Twelve landing site v2

Empowering MetaWorlds

Under construction

## Env & Conventions

- nodejs >= 12.14
- Coding convention: see `.eslintrc.json`
- CSS convention: `BEM`

## Get Started

1. `npm install`
2. `npm run start`
3. http://localhost:3000 enjoy!

## Deployment

1. `PUBLIC_URL` in `package.json`
2. `npm run build`
3. Vercel based. Direct CI / CD
