# P12 landing

## 环境和规范

-   环境要求：nodejs >= 12.14
-   技术栈：Typescript、React、ThreeJs、GreenSock
-   编码规范：见`.eslintrc.json`
-   CSS 规范：使用 `BEM`

## Get Started

1. `npm install`
2. `npm run start`
3. http://localhost:3000 enjoy!

## 打包部署

1. `package.json`中的 `scripts` 确认 `PUBLIC_URL` 环境变量
2. `npm run build`
3. 上传 build 目录
